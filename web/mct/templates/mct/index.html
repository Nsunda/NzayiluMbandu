    +Termos

    Este modulo do sistema possui os seguintes serviços:

    Solicitação da criação de termos: O usuário com perfil "Professor" pode solicitar que seja criado um termo em kikongo para um conceito cujo termo em português já é conhecido.

    Proposta de termos: O usuário com perfil "Linguista" pode propor um novo termo em kikongo.

    Certificação de termos: O usuário com perfil "Certificador" avalia as propostas de termos em kikongo para atender as solicitações dos "professores" e, em caso de ser aprovado, emite o certificado.

    Para começar a utilizar disponha das funcionalidades no menu lateral esquerdo

    Os ultimos termos certificados foram: "lista de termos"
